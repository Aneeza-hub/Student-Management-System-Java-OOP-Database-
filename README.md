# Student Management System (Java OOP Project)

## Description
This is a Java-based Student Management System developed using Swing for GUI and JDBC for Oracle database connectivity. The system allows managing students, teachers, courses, enrollments, assignments, and grades with full CRUD operations.

## Features
- Login Authentication System
- Dashboard with multiple modules
- Add, View, Update, Delete operations
- Oracle Database Integration
- User-friendly Swing GUI

## Technologies Used
- Java (OOP)
- Swing (GUI)
- JDBC
- Oracle Database

## Project Files
- Source Code
- Report (PDF)
- Presentation (PPT)

## How to Run
1. Setup Oracle database
2. Update credentials in DatabaseConnector.java
3. Compile project
4. Run WelcomePage.java
